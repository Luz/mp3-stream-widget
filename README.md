# mp3-stream-widget
An mp3 streaming widget for the Awesome Window Manager

## Changes in your rc.lua
Import command
```Lua
local mp3streamer = require("mp3-stream-widget.mp3streamer")
```
Use in the list of the widgets
```Lua
mp3streamer(),
```
